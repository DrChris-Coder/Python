 Reads words from an external file (`hangman_words.txt`)  
- Gracefully handles missing/empty files with a fallback word (`PYTHON`)  
- Displays the current state of the word with `?` placeholders  
- Validates player input (only single letters allowed)  
- Tracks incorrect attempts (max 5 by default)  
- Win/loss detection and replay functionality  
- Clear, emoji-enhanced interface for a fun CLI experience
